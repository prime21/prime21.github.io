1.数据结构

分块算法

对数列${a_n}$做如下操作:
a. 区间加法
b. 求区间中小于$c$的数

考虑分块
对于操作一：每个整块打标记，部分块，拆开暴力
对于操作二：每个整块二分，部分块，直接枚举

复杂度分析：

  1. 对于暴力重构的复杂度$O(q \log q)$
  2. 对于二分查询的复杂度$O( \frac{n}{q} \log q)$
    故平衡复杂度知$ O(q \log q) = O( \frac{n}{q} \log q ) $
    所以$O(q) = O(\sqrt n )$
    总复杂度为$O(n \sqrt n \log n ) $

左偏树

左偏树的若干性质：
记$p$点的儿子为$l$和$r$，令权值为$v$，每个点到自己内部叶子的最短距离为$h$
1.堆性质： $v_p$ 优于 $v_l$和$v_r$
2.左偏性质： $h_l \ge h_r$ 且 $h_p = h_r +1$
3.最长链性质： $ O(h_p) = O(\log size_p)$

左偏树的相关操作

  1. 合并
    仅考虑$merge(a,b)$其中$v_a$优于$v_b$
    调用$merge(a.right,b)$返回一个新的根作为a的新右子树，更新相关信息
  2. pop
    $merge(top.left,top.right)$即可
  3. 插入
    $merge(top,newnode)$jike
  4. fast-build-up
    将所有节点入队，每次取出队首$merge$然后将新树丢进队尾，仅有一棵树时结束此操作

APIO2012 dispatching
给定一棵树$T$，每个点有权值$wi$，价值$p_i$
最大化$ p_i * \sum{ q } v{q} $限制为$ \sum{q} v_q \le M $，其中$q$为$p$的子孙节点。

对于每一个点建立一个可并堆

主席树

给一个 $1 \sim n$的排列${a_n}$进行$q$次操作：
交换$a_i$和$a_j$，并求新的排列的逆序对个数。

即维护一个数据结构，满足：
1.统计 前面一段中比自己大的数，后面一段中比自己小的数
2.删除 和 插入一个新的数
故，用树状数组维护前缀权值情况，和后缀权值情况
权值情况使用主席树存储

这样，复杂度是$O(n \log^2 n)$的
