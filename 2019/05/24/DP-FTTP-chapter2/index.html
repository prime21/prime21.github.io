Definition of $\epsilon-$ DP

An algorithm $\mathcal{A}$ satisfies $\epsilon-$ DP, whiere $\epsilon \ge 0$, iff for any datasets $D$ and $D’$ that differ on one element, we have :

$ Range(\mathcal{A}) $ denotes the set of all possible outputs of the algorithm $\mathcal{A}$

Unbounded DP

$D$ and $D’$ are neighboring if $D$ can obtained from $D’$ by adding or removing one element.

Bounded DP

$D$ and $D’$ are neighboring if $D$ can obtained from $D’$ by replacing one element to another.

We use unbounded DP , beacause noting that any algorithm that satisfies $\epsilon-$ unbounded DP also satisfies $(2 \epsilon)-$ bounded DP.

Properties of $\epsilon-$ DP

  1. post-processing & sequential
  2. parallel compostion and convexity

post-processing

Given $\mathcal{A} _ 1 ( \cdot )$ that satisfies $\epsilon-$ DP, then for any algorithm $\mathcal{A} _ 2$, the compostion of $\mathcal{A} _ 1 $ and $\mathcal{A} _ 2$ , i.e. , $ \mathcal{A} _ 2 ( \mathcal{A} _ 1 ( \cdot ) ) $ satisfies $\epsilon -$ DP.
